## How to Use
copy below & paste to terminal

```bash
cd ~
git clone https://github.com/Yutarotaro/vimrc.git
cat vimrc/.vimrc >> .vimrc
mkdir -p .vim/colors
git clone https://github.com/tomasr/molokai
mv molokai/colors/molokai.vim .vim/colors
```
